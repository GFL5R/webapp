# L5R Technique Clusters

This document groups L5R techniques by mechanical similarity.
Techniques in the same cluster may represent redundant or highly
similar mechanics that should be considered when creating GFL5R
techniques to avoid duplication.

---

## Statistics

- **Total techniques:** 531
- **Clusters found:** 19
- **Unclustered (noise):** 437
- **High-similarity pairs:** 1

## High-Similarity Pairs (≥0.95 cosine similarity)

whether both need separate GFL5R analogues or if one can cover both.
